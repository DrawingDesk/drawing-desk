Drawing desk

Многопользовательское приложение для онлайн рисования

Приложение должно представлять собой одностраничное веб-приложение, которое предоставляет возможность создания собственного рабочего поля. При создании рабочего поля пользователь получит ссылку-приглашение, которую он сможет отправить любому другому пользователю, который в свою очередь перейдя по этой ссылке получит доступ к рабочему полю первого пользователя. Пользователи будут иметь возможность одновременного рисования на этом поле. Они смогут выбирать цвет и форму линий, а также смогут рисовать различные геометрические фигуры. Так же будет существовать функция “отката” изменений на произвольное количество шагов. Фактически данное приложение можно будет использовать в качестве online доски для презентации графических схем несколькими пользователями.

Функциональные требования

Согласно этим требованиям, программное средство должно предоставлять пользователю следующие функциональные возможности:

  • создания собственного рабочего поля, подключение к рабочим полям других пользователей, посредством перехода по ссылке-приглашению;
  • наличие возможности экспорта полученного изображения в одном из общепринятых форматов;
  • рисования геометрических примитивов, прямых и кривых линий на этом поле, написание текста;
  • изменения цвета, толщины, формы контуров, которым происходит рисование;
  • функции undo/redo.

Нефункциональные требования

Были сформулированы следующие нефункциональные требования:

  • изменения рабочего поля должны применяться в порядке их создания;
  • приложение должно корректно обрабатывать разрывы соединения и восстанавливать свою работу, после его восстановления.
